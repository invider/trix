Welcome to TRIX

--- Training (interactive) for UNIX ---

Your are about to start a journey through the wonderful UNIX landscape.

This world of files and pipes is at the end of your fingertips, waiting to be discovered. Take that journey and you will master the UNIX shell and look into the deepest places on the filesystem.

Rules

  • your goal is to get the highest score by completing missions
  • use available tools to monitor your status and open missions
  • to complete a mission you need to follow provided instructions and figure out the solution key
  • you can create files only inside the trix directory and your status is reflected only by the content of that directory

    Tools

    The most important tools are: 
    ./mail - shows current messages
./status - shows the overall progress
./solve - solves the mission using provided key

    How to Start

    Check out that you have curl installed. If not, install it with 

    sudo apt-get install curl
    if you're on Ubuntu/Debian or 
    sudo yum install curl
    if you're on Fedora/RedHat

    Download and run trix.install bash script

    Good luck!
